# The Lyft Ruby Gem

> A Ruby interface to the Lyft API.

## Installation

Add this line to your application's Gemfile:

    gem 'lyft', require: 'lyft'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lyft
